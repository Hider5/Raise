# Raise
![](https://img.shields.io/badge/Ruby-yellowgreen.svg)
## Installation 1
```
$git clone https://github.com/Hider5/Raise
$cd Raise
$chmod +x *
$./install.sh
$gem install bundler
$bundle install
```
## Installation 2
if install.sh error try this
```
./install2.sh
```
### Usage
```
./raise.rb -l [letter] -o [output]
sudo ./raise.rb -i [domain]
./raise.rb -f [file]
./raise.rb -a [domain]
```
